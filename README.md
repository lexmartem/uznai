# Full Stack Application

A full stack application with Next.js frontend and Spring Boot backend.

## Structure

- \`/frontend\`: Next.js application
- \`/backend\`: Spring Boot application

## Development

### Frontend

\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`

### Backend

\`\`\`bash
cd backend
./mvnw spring-boot:run
\`\`\`
